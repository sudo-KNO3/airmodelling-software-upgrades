# airmodelling-software-upgrades

Planning and audit documentation for the **AERMOD Pipeline** — an AI-powered
platform that wraps the EPA air-dispersion toolchain (AERMOD, AERMET, AERMAP,
BPIP, AERPLOT) behind a web UI and a natural-language chatbot.

This repo is the **project-scope / audit snapshot**, not the implementation:

- `PROJECT_DESCRIPTION.md` — what the platform is, the problem it solves, and
  the technology stack
- `AERMOD_AUDIT_REPORT.md` — audit findings

## The actual implementation

lives in **[aermod-pipeline](https://github.com/sudo-KNO3/aermod-pipeline)**.
Start there for the running system; this repo is kept for the design/audit
record.
